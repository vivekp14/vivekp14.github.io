# Vivek Prakash — Portfolio Website

Personal portfolio website built with HTML, Tailwind CSS, and vanilla JavaScript.

## 🌐 Live Site
👉 [vivekp14.github.io](https://vivekp14.github.io)

## 🛠️ Built With
- HTML5 & Vanilla JavaScript
- Tailwind CSS (CDN)
- Lucide Icons
- Canvas API for animated background

## ✨ Features
- Animated splash screen and scroll reveal effects
- Interactive experience accordion
- Particle network background
- Fully responsive across devices
